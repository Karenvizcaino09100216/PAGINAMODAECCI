# PAGINAMODAECCI
Proyecto web académico bilingüe sobre Moda ECCI 2026, enfocado en teoría del color, Pantone, semiología visual, sostenibilidad, economía circular e innovación tecnológica. Desarrollado con HTML5, CSS3, JavaScript y Bootstrap, aplicando principios de UX/UI, accesibilidad y diseño responsive para una experiencia moderna e interactiva.
